# GRS USA LLC 홍보 웹사이트

![GRS USA LLC](https://img.shields.io/badge/GRS-USA_LLC-blue)
![Status](https://img.shields.io/badge/Status-Complete-success)

GRS USA LLC의 친환경 토양경화 기술 및 코팅 솔루션을 홍보하는 프리미엄 웹사이트입니다.

## 🎯 프로젝트 개요

**목적**: 한국 시장 진출을 위한 핵심 마케팅 채널로서, B2G(공공기관) 및 B2B(민간기업) 고객을 타겟으로 기술력과 친환경 가치를 효과적으로 전달

**핵심 메시지**:
- 💰 **비용 절감**: 기존 공법 대비 40% 공사비 절감
- 🌱 **친환경**: 탄소 배출 70-80% 감소
- 🏆 **검증된 기술**: 4대강 사업, 여수 엑스포 레퍼런스
- 🌍 **글로벌 파트너십**: 미국 GRS USA LLC의 기술력

## 🎨 디자인 시스템

### 컬러 팔레트
```css
--color-primary-deep: #1a365d    /* 신뢰, 전문성 */
--color-primary-blue: #2563eb
--color-secondary-forest: #2f855a /* 친환경 */
--color-secondary-green: #10b981
--color-accent-orange: #ed8936    /* 혁신, 에너지 */
```

### 타이포그래피
- **한글**: Noto Sans KR
- **영문**: Inter
- **크기**: 12px ~ 60px (반응형)

### 간격 시스템
- 4px ~ 96px (Spacing 변수 활용)

### 그림자
- sm, md, lg, xl, 2xl (5단계)

### Border Radius
- sm (4px) ~ 2xl (24px)

## 🚀 사용 방법

### 1. 로컬 실행
```bash
# 파일 탐색기에서 index.html을 더블클릭하거나
# 브라우저에서 직접 엽니다
file:///경로/grs/index.html
```

### 2. Live Server 사용 (권장)
```bash
# VS Code의 Live Server 확장 설치 후
# index.html에서 우클 릭 → Open with Live Server
```

### 3. 간단한 HTTP 서버
```bash
# Python 3
python -m http.server 8000

# Node.js (npx 사용)
npx http-server

# 브라우저에서 localhost:8000 접속
```

## 💡 주요 기능 사용법

### 비용 절감 계산기
1. `contact.html` 페이지로 이동
2. 시공 면적(m²) 입력
3. 프로젝트 유형 선택 (도로, 산책로, 물류센터, 주차장)
4. "계산하기" 버튼 클릭
5. 예상 견적 및 절감액, 탄소 감축량 확인

### 문의 폼
1. 필수 정보 입력 (이름, 회사명, 이메일, 전화번호)
2. 문의 유형 선택
3. 상세 내용 작성
4. 개인정보 동의 체크
5. "문의 전송" 버튼 클릭

## 🔧 커스터마이징

### 색상 변경
`css/main.css`의 `:root` 섹션에서 CSS 변수를 수정하세요.

```css
:root {
  --color-primary-blue: #YOUR_COLOR;
  --color-secondary-green: #YOUR_COLOR;
  /* ... */
}
```

### 폰트 변경
`<head>` 섹션의 Google Fonts 링크와 CSS 변수를 수정하세요.

### 콘텐츠 수정
각 HTML 파일에서 텍스트, 이미지, 링크 등을 직접 수정할 수 있습니다.
